## Installation

### Step 1: Verify Python Version

```bash

# Should show: Python 3.13.x or higher
```

If Python 3.13+ is not installed:
- Windows: Download from [python.org](https://www.python.org/downloads/)
- macOS: `brew install python@3.13`
- Linux: `sudo apt install python3.13` (or equivalent)

```bash
uv --version
# Should show: uv 0.6.x or higher
```

If UV is not installed:

**Windows (PowerShell)**:
```powershell

```

**macOS/Linux**:
```bash

```

## Data Persistence

### Important: No Data Persistence in Phase I

‚ö†Ô∏è **All tasks are stored in memory only.**

- Data is **lost when the application exits**
- No database or file storage
- This is intentional for Phase I simplicity

**Example**:
```
1. Add tasks: Task #1, Task #2, Task #3
2. Exit application
3. Run application again
4. View tasks: "No tasks found" (all previous tasks lost)
```

- Phase III+: Cloud storage with sync

## Troubleshooting

### Common Issues

#### Issue: `ModuleNotFoundError: No module named 'src'`

**Solution**:
```bash
# Make sure you're in the repository root
cd E:\hackathon-02\Todo-App

# Run with explicit Python path
uv run python src/main.py
```

#### Issue: `SyntaxError` or `IndentationError`

**Cause**: Python version < 3.13 or file corruption

**Solution**:
```bash
# Verify Python version
python --version

# If < 3.13, upgrade Python
# Then re-run
uv run src/main.py
```

#### Issue: `UnicodeDecodeError` when entering emojis

**Cause**: Terminal encoding not set to UTF-8

**Solution**:

**Windows Command Prompt**:
```cmd
chcp 65001
uv run src/main.py
```

**PowerShell**:
```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
uv run src/main.py
```

**Linux/macOS**: Should work by default (UTF-8 is standard)

#### Issue: Menu doesn't clear/scroll is messy

**Expected Behavior**: Output scrolls upward (no screen clearing)

Phase I intentionally does **not** clear the screen between operations. This is acceptable for simplicity. Screen clearing can be added in Phase II.

#### Issue: Tasks disappear after exit

**Expected Behavior**: This is correct for Phase I (in-memory only)

See "Data Persistence" section above. Data loss on exit is intentional.
